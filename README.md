# IPerl
> Interactive Perl interpreter

This is a simple script that aims to provide an easy and fun way to run a interactive Perl interpreter.

![](header.png)

## Installation

OS X & Linux:

```sh
sudo cp iperl.pl /bin/iperl
```

Windows:

```sh
cp iperl.pl %HOMEPATH%
```

## Usage example

```
user@pc:~$ iperl
IPerl> 
```
